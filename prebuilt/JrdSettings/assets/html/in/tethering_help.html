Tether USB

Anda dapat menyambungkan ponsel ke komputer menggunakan kabel USB untuk berbagi sambungan Internet ponsel dengan komputer
  • Penyambungan berfungsi di Windows Vista, Windows 7, dan Linux

  • Anda tidak dapat memasang kartu SD ponsel di komputer bila USB tersambung

  • Untuk informasi lebih lanjut, termasuk cara menggunakan penyambungan USB dengan OS lainnya, kunjungi http://www.android.com/tether#usb

Tethering Bluetooth

Anda dapat menghubungkan perangkat Android ke komputer via sambungan Bluetooth untuk berbagi sambungan Internet perangkat dengan komputer
  • Tethering Bluetooth dapat diterapkan untuk komputer apapun atau perangkat lain yang dapat memperoleh sambungan Internet via Bluetooth

  • Untuk informasi lebih lanjut, kunjungi http://www.android.com/tether#bluetooth

Hotspot Wi-Fi portabel

Anda dapat menggunakan ponsel sebagai hotspot Wi-Fi portabel untuk berbagi sambungan Internet ponsel dengan satu atau beberapa komputer maupun perangkat lainnya
  • Anda dapat mengkonfigurasi hotspot dengan pengaturan hotspot Wi-Fi

  • Untuk informasi lebih lanjut, kunjungi http://www.android.com/tether#wifi
