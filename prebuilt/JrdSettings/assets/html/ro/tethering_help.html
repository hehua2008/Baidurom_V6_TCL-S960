Modem USB

Puteţi să conectaţi dispozitivul Android la computer cu un cablu USB, pentru a partaja conexiunea la Internet a dispozitivului cu computerul
  • Legarea USB funcţionează cu Windows Vista, Windows 7, şi Linux

  • Dacă dispozitivul are o cartelă SD sau un dispozitiv de stocare USB, nu îl puteţi ataşa la computer când funcţia de modem USB este activată

  • Pentru informaţii suplimentare, vizitaţi adresa http://www.android.com/tether#usb

Funcţie de modem Bluetooth

Puteţi conecta dispozitivul Android la computer printr-o conexiune Bluetooth, pentru a partaja conexiunea la Internet a dispozitivului cu computerul
  • Funcţia de modem Bluetooth funcţionează cu orice computer sau un alt dispozitiv care poate obţine o conexiune la Internet prin Bluetooth

  • Pentru informaţii suplimentare, vizitaţi adresa http://www.android.com/tether#bluetooth

Hotspot Wi-Fi portabil

Puteţi să transformaţi dispozitivul Android într-un hotspot Wi-Fi portabil, pentru a partaja conexiunea la Internet a dispozitivului Android cu unul sau mai multe computere sau alte dispozitive
  • Când dispozitivul serveşte drept hotspot Wi-Fi, nu puteţi utiliza aplicaţiile dispozitivului pentru a accesa Internetul prin conexiunea Wi-Fi (dar rămâneţi conectat la Internet prin reţeaua mobilă de date)

  • Configuraţi hotspotul cu setările de hotspot Wi-Fi

  • Pentru informaţii suplimentare, vizitaţi adresa http://www.android.com/tether#wifi
