USB เทเธอริ่ง

คุณสามารถเทเธอร์โทรศัพท์ของคุณไปยังคอมพิวเตอร์ของคุณด้วยสายเคเบิล USB, เพื่อแชร์การเชื่อมต่ออินเทอร์เน็ตของโทรศัพท์ของคุณกับคอมพิวเตอร์ของคุณ
  • คุณสมบัติเทเธอริ่ง ทำงานกับ Windows Vista, Windows 7 และ Linux

  • คุณไม่สามารถเมาต์ SD การ์ดของโทรศัพท์ของคุณบนคอมพิวเตอร์ได้ในขณะที่เทเธอร์ผ่าน USB อยู่

  • สำหรับข้อมูลเพิ่มเติม ที่ประกอบด้วยการใช้ USB เทเธอริ่งกับ OS อื่นๆ ให้ดู http://www.android.com/tether#usb

การแชร์เน็ตผ่านบลูทูธ

คุณสามารถต่ออุปกรณ์แอนดรอยด์ของคุณเข้ากับคอมพิวเตอร์ได้ทางบลูทูธ เพื่อแชร์การเชื่อมต่อเน็ตของอุปกรณ์ของคุณกับคอมพิวเตอร์
  • การแชร์เน็ตผ่านบลูทูธใช้ได้กับคอมพิวเตอร์ใด ๆ หรืออุปกรณ์อื่นที่สามารถรับการเชื่อมต่อเน็ตผ่านบลูทูธได้

  • สำหรับข้อมูลเพิ่มเติม โปรดเยี่ยมชมที่ http://www.android.com/tether#bluetooth

ฮ็อตสป็อต Wi-Fi พกพา

คุณสามารถเปลี่ยนโทรศัพท์ของคุณไปเป็นฮ็อตสป็อต Wi-Fi พกพา เพื่อแชร์การเชื่อมต่ออินเทอร์เน็ตของโทรศัพท์ของคุณกับคอมพิวเตอร์หรืออุปกรณ์อื่นๆ ตั้งแต่หนึ่งเครื่องขึ้นไปได้
  • เมื่อโทรศัพท์ของคุณทำหน้าที่เป็นฮ็อตสป็อต Wi-Fi, คุณจะไม่สามารถใช้แอปพลิเคชั่นของโทรศัพท์ของคุณเพื่อเข้าถึงอินเทอร์เน็ตผ่านการเชื่อมต่อ Wi-Fi ของเครื่องได้

  • คุณกำหนดค่าฮ็อตสป็อตด้วยการตั้งค่าฮ็อตสป็อต Wi-Fi

  • สำหรับข้อมูลเพิ่มเติม โปรดเยี่ยมชมที่ http://www.android.com/tether#wifi
