USB-Tethering

Sie können Ihr Telefon über ein USB-Kabel mit Ihrem Computer verbinden, um die Internetverbindung auf Ihrem Telefon auf Ihrem Computer nutzen zu können.
  • Tethering funktioniert mit Windows Vista, Windows 7 und Linux.

  • Sie können die SD-Karte Ihres Telefons nicht in Ihren Computer einsetzen, solange eine USB-Verbindung besteht.

  • Weitere Informationen (einschließlich USB-Tethering mit anderen Betriebssystemen) finden Sie auf http://www.android.com/tether#usb.

Bluetooth-Tethering

Sie können Ihr Android-Gerät per Tethering über eine Bluetooth mit Ihrem Computer verbinden und die Internetverbindung Ihres Gerätes mit Ihrem Computer teilen
  • Bluetooth-Tethering funktioniert mit jedem Computer bzw. anderen Gerät, das eine Internetverbindung via Bluetooth beziehen kann

  • Weitere Informationen erhalten Sie auf http://www.android.com/tether#bluetooth

WLAN-Tethering

Sie können Ihr Telefon in einen WLAN-Zugangspunkt verwandeln, um die Internetverbindung Ihres Telefons auf einem oder mehreren Computern bzw. anderen Geräten nutzen zu können.
  • Wenn Ihr Telefon als Zugangspunkt fungiert, können Sie die darauf installierten Anwendungen nicht verwenden, um über die WLAN-Verbindung auf das Internet zuzugreifen.

  • Sie konfigurieren einen WLAN-Zugangspunkt in den WLAN-Tethering-Einstellungen.

  • Weitere Informationen erhalten Sie auf http://www.android.com/tether#wifi.
