Bluetooth-Tethering

Sie können Ihr Android-Gerät per Tethering über eine Bluetooth mit Ihrem Computer verbinden und die Internetverbindung Ihres Gerätes mit Ihrem Computer teilen
  • Bluetooth-Tethering funktioniert mit jedem Computer bzw. anderen Gerät, das eine Internetverbindung via Bluetooth beziehen kann

  • Weitere Informationen erhalten Sie auf http://www.android.com/tether#bluetooth
