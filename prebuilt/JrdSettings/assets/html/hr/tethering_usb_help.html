USB dijeljenje

Možete dijeliti svoj Android uređaj s računalom putem USB kabela kako biste podijelili svoju Internet vezu
  • USB dijeljenje radi s Windows Vista, Windows 7 i Linux operacijskim sustavima

  • Ukoliko vaš uređaj ima SD karticu ili USB pohranu, prilikom USB dijeljenja im ne možete pristupiti

  • Za više informacija, uključujući USB dijeljenje s drugim operacijskim sustavima, pogledajte http://www.android.com/tether#usb
