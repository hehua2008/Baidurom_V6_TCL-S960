USB dijeljenje

Možete dijeliti svoj Android uređaj s računalom putem USB kabela kako biste podijelili svoju Internet vezu
  • USB dijeljenje radi s Windows Vista, Windows 7 i Linux operacijskim sustavima

  • Ukoliko vaš uređaj ima SD karticu ili USB pohranu, prilikom USB dijeljenja im ne možete pristupiti

  • Za više informacija, uključujući USB dijeljenje s drugim operacijskim sustavima, pogledajte http://www.android.com/tether#usb

Bluetooth dijeljenje

Možete dijeliti svoj Android uređaj s računalom putem Bluetooth-a kako biste podijelili svoju Internet vezu
  • Bluetooth dijeljenje radi s bilo kojim računalom ili uređajem koji može prihvatiti Internet vezu putem Bluetooth-a

  • Za više informacija pogledajte http://www.android.com/tether#bluetooth

Prijenosni Wi-Fi hotspot

Svoj Android uređaj možete pretvoriti u prijenosni Wi-Fi hotspot kako biste podijelili svoju Internet vezu s jednim ili više računala ili uređaja
  • Kad je vaš telefon aktivan kao prijenosni Wi-Fi hotspot, ne možete pristupati Internetu putem Wi-Fi veze (naravno, pristup Internetu imate putem podatkovne mobilne veze)

  • Unutar Wi-Fi hotspot postavki možete postaviti svoju vezu

  • Za više informacija, posjetite http://www.android.com/tether#wifi
