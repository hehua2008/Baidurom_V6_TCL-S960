USB टीथरिंग

अपने फ़ोन के इंटरनेट कनेक्शन को अपने कंप्यूटर के साथ साझा करने के लिए, आप USB केबल के द्वारा अपने फ़ोन को कंप्यूटर से टीथर कर सकते हैं
  • टीथरिंग Windows Vista, Windows 7, और Linux के साथ काम करती है

  • USB टीथर्ड होने पर आप अपने फ़ोन के SD कार्ड को अपने कंप्यूटर पर माउंट नहीं कर सकते हैं

  • USB टीथरिंग के दूसरे ऑपरेटिंग सिस्टम्स के साथ उपयोग के बार में जानकारी सहित अन्य जानकारी के लिए, http://www.android.com/tether#usb देखें
