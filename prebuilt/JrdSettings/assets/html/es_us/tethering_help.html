Modem USB

Puede conectar el teléfono a su ordenador con un cable USB para compartir la conexión a Internet del teléfono con el equipo.
  • La conexión funciona con Windows Vista, Windows 7 y Linux.

  • Puede montar la tarjeta SD del teléfono en el equipo una vez realizada la conexión USB.

  • Para obtener más información, incluido el uso como modem USB con otros sistemas operativos, consulte la página http://www.android.com/tether#usb.

Bluetooth instalación

Tú puedes instalar tu dispositivo Android en tu ordenador vía Bluetooth, para compartir tus dispositivos con tu ordenador a través de la conexión de Internet.
  • Bluetooth funciona con cualquier ordenador o cualqiuier otro dispositivo que pueda tener conexión a Internet vía Bluetooth.

  • Para más información, vea http://www.android.com/tether#bluetooth.

Datos Wi-Fi

Puede convertir su teléfono en un punto de acceso Wi-Fi para compartir la conexión a Internet del teléfono con uno o con varios equipos o dispositivos.
  • Cuando el teléfono actúe como un punto de acceso, no podrá utilizar sus aplicaciones para acceder a Internet a través de la conexión Wi-Fi del dispositivo

  • Para configurar un punto de acceso Wi-Fi, acceda a la configuración de Zona Wi-Fi.

  • Para obtener más información, consulte la página http://www.android.com/tether#wifi
