Портативна точка доступу Wi-Fi

Для того, щоб надати спільний доступ до Інтернету одному або декільком комп'ютерам чи іншим пристроям, можна підключити пристрій в ОС Android як портативну точку доступу Wi-Fi
  • У разі використання пристрою як портативної точки доступу Wi-Fi не можна скористатися програмами для отримання доступу до Інтернету через підключення Wi-Fi на пристрої (однак, пристрій залишатиметься підключеним до Інтернету через мобільну мережу для передавання даних)

  • Можна настроїти точку доступу за допомогою установок точки доступу Wi-Fi

  • Додаткові відомості див. на http://www.android.com/tether#wifi
