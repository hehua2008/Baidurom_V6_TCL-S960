Модем USB

Можна підключити пристрій з ОС Android до комп'ютера за допомогою кабелю USB для отримання спільного доступу до Інтернету на комп'ютері
  • Модем USB може працювати з операційними системами Windows Vista, Windows 7 і Linux

  • Неможливо підключити пристрій до комп'ютера під час підключення модема USB, якщо на ньому використовується карта пам'яті або USB-накопичувач

  • Додаткові відомості, включаючи використання модема USB з іншими операційними системами, див http://www.android.com/tether#usb
