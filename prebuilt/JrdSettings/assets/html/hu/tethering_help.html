USB kapcsolat-megosztás

Csatlakoztathatja Android eszközét a számítógéphez USB-kábellel az eszköz internetkapcsolatának megosztásához a számítógéppel
  • USB megkötést együttműködik a Windows Vista, Windows 7 és Linux

  • Ha eszköze SD-kártyával vagy USB-tárolóval rendelkezik, nem csatlakoztathatja a számítógéphez USB internetkapcsolat-megosztás használata közben

  • További információkért látogasson el ide http://www.android.com/tether#usb

Bluetooth kapcsolat-megosztás

Csatlakoztathatja Android eszközét a számítógéphez Bluetooth-kapcsolaton az eszköz internetkapcsolatának megosztásához a számítógéppel
  • A Bluetooth internet-megosztás bármilyen számítógépen vagy egyéb eszközzel működik, amely internetkapcsolatot képes létesíteni Bluetooth-kapcsolaton

  • További információkért látogasson el ide http://www.android.com/tether#bluetooth

Hordozható Wi-Fi hotspot

Android eszközét hordozható Wi-Fi hotspottá változtathatja, és megoszthatja Android eszközének internetkapcsolatát egy vagy több számítógéppel vagy egyéb eszközzel
  • Ha eszköze Wi-Fi hotspotként működik, nem használhatja eszköze alkalmazásait az internet eléréséhez a Wi-Fi kapcsolatán (azonban természetesen tovább kapcsolódik az internethez a mobil adathálózaton keresztül)

  • A hotspotot konfigurálhatja a Wi-Fi hotspot beállítások menü alatt

  • További információkért látogasson el ide http://www.android.com/tether#wifi
