Módem USB

Puede conectar el teléfono a su ordenador con un cable USB para compartir la conexión a Internet del teléfono con el ordenador.
  • La conexión funciona con Windows Vista, Windows 7 y Linux.

  • Puede instalar la tarjeta SD del teléfono en el ordenador una vez realizada la conexión USB.

  • Para obtener más información, incluido el uso como módem USB con otros sistemas operativos, consulte la página http://www.android.com/tether#usb.
