Módem USB

Puede conectar el teléfono a su ordenador con un cable USB para compartir la conexión a Internet del teléfono con el ordenador.
  • La conexión funciona con Windows Vista, Windows 7 y Linux.

  • Puede instalar la tarjeta SD del teléfono en el ordenador una vez realizada la conexión USB.

  • Para obtener más información, incluido el uso como módem USB con otros sistemas operativos, consulte la página http://www.android.com/tether#usb.

Módem Bluetooth

Puede conectar su dispositivo Android a su ordenador mediante una conexión Bluetooth, para compartir la conexión a Internet de dicho dispositivo con su ordenador
  • El módem Bluetooth funciona con cualquier equipo u otro dispositivo que pueda obtener una conexión a Internet a través de la funcionalidad Bluetooth

  • Para obtener más información, consulte la página http://www.android.com/tether#bluetooth

Conexión Wi-Fi

Puede convertir su teléfono en un punto de acceso Wi-Fi para compartir la conexión a Internet del teléfono con uno o con varios equipos o dispositivos.
  • Cuando el teléfono actúe como un punto de acceso, no podrá utilizar sus aplicaciones para acceder a Internet a través de la conexión Wi-Fi del dispositivo

  • Para configurar un punto de acceso Wi-Fi, acceda a la configuración de Zona Wi-Fi.

  • Para obtener más información, consulte la página http://www.android.com/tether#wifi.
